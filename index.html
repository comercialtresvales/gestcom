🔥 Firebase Realtime Database
Grátis · Sem cartão
Banco em tempo real do Google, 100% gratuito e sem precisar de cartão de crédito. Qualquer alteração feita por um usuário aparece para todos os outros em até 5 segundos.
Encontrada no console Firebase → Realtime Database
📋 Como configurar (2 minutos)
1No Firebase Console → Bancos de dados → Realtime Database
2Copie a URL que aparece:
    https://sistematresvales-default-rtdb.firebaseio.com
3Cole no campo "URL do Realtime Database" ao lado
4Clique em Conectar
5Clique em 📤 Enviar dados para nuvem
6Compartilhe o arquivo HTML — todos conectam na mesma URL → sincronizado! ✅
💡 Dica de segurança: Após conectar, vá em Regras no Realtime Database e defina uma senha ou limite o acesso por data para proteger os dados.
⚙️ Comportamento da Sincronização
Ativado — salva automaticamente
Ativado — atualiza automaticamente
💡 Enviar sobrescreve os dados da nuvem com os dados locais atuais. Carregar sobrescreve os dados locais com os da nuvem. Com auto-salvar ativado, cada cadastro/edição é enviado automaticamente.
